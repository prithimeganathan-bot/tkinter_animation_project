# Tkinter Animation Project

A simple Python project demonstrating animation using Tkinter. This project showcases moving objects (spaceship/UFO) on a canvas, controlled by basic physics-like logic for smooth animation.

## Features

- Animated spaceship/UFO on a Tkinter canvas.
- Smooth movement with velocity control.
- Background image support.
- Easy to extend for additional animations or game logic.
- Lightweight and beginner-friendly.

## Technologies Used

- **Python 3.x**
- **Tkinter** for GUI and animation
- **PIL (Pillow)** for image handling
